### Icloud
Dung Imap

Vào appleid.apple.com → đăng nhập.

Đảm bảo đã bật Two-Factor Authentication.

Trong phần Sign-In and Security → App-Specific Passwords:

Chọn Generate Password...

Đặt tên (ví dụ node-imap) → Apple trả về 16 ký tự (vd: abcd-efgh-ijkl-mnop).

Đó là password bạn dùng cho IMAP (không dùng password Apple ID bình thường).
